Lesson 1.2: Explicit and Recursive Functions

In mathematics, a sequence can be defined in two common ways:

1. Explicit Formula

An explicit formula defines the nth term using n directly.
Example: F(n) = 3n + 1

2. Recursive Formula

A recursive formula defines each term using the previous term.
Example: F(1) = 5 , and F(n) = F(n-1) + 3

These are examples of Arithmatic functions there are also Geometric functions, but those aren't important right now.
